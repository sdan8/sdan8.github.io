VRP基础

发表于 2019-12-13 | 更新于: 2019-12-13 | 分类于 网络 | 评论数 | 热度: ℃

交换机隔离冲突域，路由器隔离广播域，随着接入到网络中的终端数量增加，网络设备负担越来越重，这时网络设备可以通过华为专有的VRP系统来提升运行效率。

VRP基础

通用路由平台（Versatile Routing Platform，VRP）是华为数据通信产品的通用操作系统平台，它以IP业务为核心，采用组件化的体系结构，在实现丰富功能特性的同时，还提供了基于应用的可裁剪和可扩展的功能，使路由器和交换机的运行效率大大增加。

VRP的发展

VRP5是分布式操作系统，具有高可靠性，高性能，绝大多数产品使用VRP5。少数高性能设备，如NE系列路由器使用VRP8。

设备管理接口

console接口

通过console线，一端接到设备console接口上，另一端接到PC上，如果是笔记本，可以加一条USB和RS232转接线。在PC上通过超级终端或者SecureCRT等软件进行访问。

miniUSB接口

一端接到设备的miniUSB接口上，另一端接到PC上，PC上需要安装miniUSB驱动，在华为官网进行下载，但是驱动仅支持Windows XP、Vista、7的系统，有很大局限性，故推荐使用console接口对设备进行配置。

命令行基础

设备初始化启动

开机界面提供了正在运行VRP版本等信息，启动完成后，系统提示目前正在运行的是自动配置模式，询问用户是否退出自动配置模式，如果选择手动配置，则输入Y。一般情况下选择手动配置。

命令行视图

用户视图

<Huawei>，采用尖括号表示的为用户视图，只能进行一些基本操作，查看设备状态等。

系统视图

在用户视图下输入system-view命令，可以进入系统视图。此时显示为[Huawei]，可以配置系统参数等，一般配置都需要进入系统视图。

接口视图

进入设备的接口下进行配置使用的。例如在系统视图下输入[Huawei]interface GigabitEthernet 0/0/0命令，来进入g 0/0/0接口，此时显示为[Huawei-GigabitEthernet0/0/0]，可以配置接口参数，IP地址等。

协议视图

在系统视图下进入该视图，可以配置该协议的相应内容。

配置文件管理

  • Current-Configuration File 当前配置文件，存放在RAM中。
  • Saved-Configuration File 保存配置文件，存放在Flash/SD卡中，通常用.cfg或.zip作为扩展名。

设备启动时，会加载保存配置文件到RAM中，作为当前配置文件。

存储设备修复

当存储设备文件系统出现异常时，可以通过 fixdisk 命令进行修复。如果使用该命令修复后，仍然收到系统建议修复的信息，则表示物理介质可能已经损坏。

当文件系统异常无法修复时，并且确认不再需要存储器上的所有数据时，可使用 format 命令格式化存储设备。格式化会导致所有文件丢失，并且不能恢复。

VRP系统管理

随着VRP版本的更新，支持的特性也不同。当用户需要对VRP进行版本更新时，可通过以下步骤进行。

更新前需要对日志文件、配置文件进行备份，防止更新失败导致数据丢失。

网络设备可以从服务器获取VRP系统文件，如果本地有VRP系统文件，可以通过USB接口连接电脑直接进行更新。

文件传输协议

FTP采用两个TCP连接，控制连接和数据连接。控制连接用于连接控制端口，传输控制命令；数据连接用于连接数据端口，传输数据。ARG3系列路由器既可以作为FTP客户端，又可以作为服务端。缺省情况下，AR2200采用主动模式建立数据连接。

TFTP是简化版的FTP，采用UDP进行文件传输。ARG3系列路由器只可以作为TFTP客户端。

更新VRP

  1. 配置IP地址，保证客户端和服务器的连通性。
  2. 查看客户端的剩余空间。通过dir命令。当剩余存储空间不足时，可以删除无关的VRP文件或其他文件来释放空间。
  3. 通过FTP远程登录到服务器上。命令 ftp 10.1.1.2，远程登录到IP为10.1.1.2的ftp服务器上。
  4. 输入用户名和密码正确后，可以下载ftp服务器上的更新文件。[ftp] get AR2220E-V200R007C00SPC600.cc

如果采用TFTP，可以直接下载文件。

  1. 设置下次启动时加载下载的VRP文件。
  1. 使用 display startup 命令来查看设置是否成功。
  2. 使用 reboot 命令进行重启设备。
