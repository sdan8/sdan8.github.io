Windows安装MySQL5.7

发表于 2019-11-11 | 更新于: 2019-11-11 | 分类于 MySQL | 评论数 | 热度: ℃

重装系统后，需要配置环境，记录下安装MySQL5.7的过程。

  • 操作系统：Windows 10 x64位
  • MySQL版本：5.7.28

卸载原版本

由于是重装系统，所以电脑内并没有安装过MySQL，这里忽略卸载过程。如果电脑中有MySQL服务，建议卸载原版本。

下载MySQL5.7.28

下载地址
https://dev.mysql.com/downloads/mysql/

进入该网址后，选择版本进行下载。

根据自己操作系统选择相应版本进行下载，这里下载64位。

解压安装包并配置环境变量

将下载好的压缩包进行解压到想要安装的位置。这里我放在了 C:\code 下。

配置环境变量：

右击“此电脑”，点击“属性”，打开“高级系统设置”，点击“环境变量”。



  • 变量名：MYSQL_HOME
  • 变量值：C:\code\mysql-5.7.28-winx64

此处的变量值应为MySQL解压的目录路径

选择Path变量，新建值 %MYSQL_HOME%\bin

创建数据目录及配置文件

在安装MySQL的目录下创建data目录、Uploads目录和my.ini配置文件。
配置文件内容如下，根据实际情况修改配置文件。

此时是我们的目录结构。

初始化数据库并启动服务

打开命令提示符（cmd）进入MySQL安装目录下的bin目录，执行mysqld --initialize-insecure

异常

如果可以正常执行命令，则跳过这一步。

此时由于我电脑中缺少vcredist，会弹出提示如下：

需要下载vcredist，链接如下：

https://www.microsoft.com/zh-CN/download/details.aspx?id=40784

打开页面后选择语言，点击下载，选择要下载的程序，这里我选择vcredist_x64.exe，点击next，开始下载。

下载好之后直接运行安装即可。此时再运行初始化命令继续以下步骤。

继续

执行mysqld -install MySQL安装服务，这里的MySQL是服务名称，可以修改。

执行net start MySQL启动MySQL服务。此处MySQL名称要和上面一致。

修改密码

打开data\error.log日志，查看密码提示。

说明密码默认为空。输入mysql -uroot登录，修改密码。

图中所示，第一条命令为登录；第二条命令为修改密码，其中，123456为设置的密码；第三天命令为刷新权限表。

到此为止我们的MySQL就安装配置完毕，可以用Navicat或者MySQL-Front等工具登录使用了。
