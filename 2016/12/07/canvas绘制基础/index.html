canvas绘制基础

发表于 2016-12-07 | 分类于 JavaScript |

创建canvas

Html部分

  1. 添加canvas标签，指定id属性便于JS获取到这个元素
  2. 在canvas标签中添加文字，当浏览器不支持canvas标签时，便会显示该文字，若浏览器支持canvas，则隐藏文字内容。
  3. 设置canvas的样式，便于看到canvas区域

js部分

  1. 根据id获取canvas元素
  2. 通过 canvas.width 和 canvas.height 设置canvas画布的大小
  3. 通过 canvas.getContext('2d') 方法获取context
  4. 对context操作，绘制图形

绘制图形

绘制直线

  1. 通过 context.beginPath() 方法来开始一条路径，或重置当前路径
  2. context.moveTo(x, y) 方法来把路径移动到 (x, y) ，不创建线条
  3. context.lineTo(x, y) 方法来添加一个新的点，然后在画布中创建从该点到最后指定点的线条
  4. 使用 context.lineWidth 设置或返回当前线条的宽度
  5. 使用 context.strokeStyle 设置或返回用于笔触的颜色、渐变或模式
  6. 最后通过 context.stroke() 方法绘制已定义的路径
  7. 如果要创建多条线条，通过 context.lineTo(x, y) 方法来顺序添加点

填充

  1. 绘制路径，与绘制直线相同
  2. 通过 context.closePath() 创建从当前点回到起始点的路径，使路径闭合
  3. 通过 fillStyle 设置或返回用于填充绘画的颜色、渐变或模式
  4. 通过 context.fill() 方法填充

绘制弧线

通过 context.arc()方法来绘制弧线

语法

context.arc(x, y, r, sAngle, eAngle, counterclockwise)

参数 描述
x 圆的中心的 x 坐标
y 圆的中心的 y 坐标
r 圆的半径
sAngle 起始角，以弧度计。（弧的圆形的三点钟位置是 0 度）
eAngle 结束角，以弧度计
counterclockwise 可选。规定应该逆时针还是顺时针绘图。False = 顺时针，true = 逆时针
  1. 弧度三点钟位置是 0 度，然后按照顺时针方向，6点钟是 0.5*PI ，9点钟是 1*PI ，12点钟是 1.5*PI ，3点钟是 2*PI
  2. 不论 counterclockwise 参数设置为 true 还是 false ，弧度表示位置始终不变
