'Python之常用内置函数'

发表于 2017-02-08 | 更新于: 2017-02-08 | 分类于 Python | 评论数 | 热度: ℃

列举常用的内置函数

  • abs() 取绝对值
  • max() 取最大值
  • min() 取最小值
  • len() 取长度
  • divmod(x, y) 取商和模，返回一个tuple (x//y, x%y)
  • pow(x, y[, z]) 求幂,x的y次幂,z表示求余
  • round() 四舍五入
  • callable() 测试函数是否可以被调用
  • isinstance() 判断类型是否为指定类型
  • cmp(x, y) 比较x和y的大小，如果 x>y 返回 1 ，如果 x<y 返回 -1，如果 x==y 返回 0
  • range()
  • xrange()
  • type() 查看对象类型
  • int() 转换成整型
  • long() 转换成长整型
  • float() 转换成浮点型
  • complex()
  • str()
  • list()
  • tuple()
  • hex()
  • oct()
  • chr() 将ASCII码转换成字符
  • ord() 将字符转换成ASCII码

string函数

  • str.capitalize() 首字母大写
  • str.replace() 返回替换的字符串
  • str.split() 切割

序列处理函数

  • len()
  • max()
  • min()
  • filter()
  • zip()
  • map()
  • reduce()
